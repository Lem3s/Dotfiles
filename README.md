Pacotes Utilizados/Necessários: yay, cargo, npm, bat, exa, ohmyzsh, neofetch, htop, unzip, neovim.

wget https://github.com/ryanoasis/nerd-fonts/releases/download/v3.0.2/JetBrainsMono.zip            .
